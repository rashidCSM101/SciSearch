Search Help

Learn how to use our search engine effectively

Basic Search

To perform a basic search, simply enter your search terms in the search box and press Enter or click the Search button.

The search engine will return documents that contain all of the words in your query.

Example: machine learning

This will return documents that contain both "machine" and "learning".

For more precise searches, try using the advanced search options or boolean operators.

Boolean Operators

Our search engine supports the following boolean operators:

  • AND: Returns documents that contain both terms.
  • OR: Returns documents that contain either term.
  • NOT: Excludes documents that contain the specified term.
Examples:
neural AND networks
machine OR learning
algorithm NOT genetic

You can also use parentheses to group terms and create more complex queries:

Example: (neural OR deep) AND networks NOT convolutional

Boolean operators must be in ALL CAPS (AND, OR, NOT) to be recognized as operators.

Advanced Search

The advanced search page allows you to create complex queries using separate fields for AND, OR, and NOT operators.

  • AND terms: Documents must contain all of these terms.
  • OR terms: Documents must contain at least one of these terms.
  • NOT terms: Documents must not contain any of these terms.

This is particularly useful for creating complex queries without having to remember the syntax for boolean operators.

You can access the advanced search by clicking the "Advanced Boolean Search" link on the home page.

Spell Correction

Our search engine includes spell correction functionality. If you misspell a search term, the system will suggest a correction.

For example, if you search for "nueral networks", the system will suggest "neural networks" as a correction.

You can click on the suggested correction to perform a new search with the corrected terms.

Spell correction works best for common technical terms in the field of computer science and related disciplines.

PDF Viewing

For each document in our database, you can view the corresponding PDF file (if available) by clicking the "View Document" button.

The PDF will open in a new tab, allowing you to read the full paper directly in your browser.

You can also download the PDF for offline reading by using your browser's download functionality.

If a PDF is not available for a particular document, you will see an error message.
